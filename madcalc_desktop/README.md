# MadCalc Desktop

Desktopowa wersja `MadCalc` dla `macOS` i `Windows`, zbudowana we Flutterze.

## Funkcje

- działanie całkowicie offline
- przełączanie jednostek `cm / mm`
- dodawanie elementów `długość + ilość`
- ustawienie długości sztangi i grubości piły
- generowanie planu cięcia z odpadem i wykorzystaniem materiału
- własne nazwy dla sztang
- zapis raportu PDF na dysk
- lokalna pamięć ostatniego stanu

## Uruchamianie

### macOS

```bash
flutter run -d macos
```

### Windows

```bash
flutter run -d windows
```

## Build

### macOS

```bash
flutter build macos
```

### Windows

```bash
flutter build windows
```
